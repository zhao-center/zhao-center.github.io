「Sony Xperia」內置應用刪除

前言

Sony的Xperia系列是一個非常好的手機系列，因為這個系列中的大部分手機都支持「Android Open Source Project (AOSP)」而且其本身在Github上的開源項目就非常豐富。雖然對於筆者這種追求軟件自由的人來說，還是有很多不如意的地方，但是這在如今的大型手機廠商中算是一股清流了。加上Sony自身獨有的信仰屬性，真的是手機中的不二選擇。如今，「Xperia ZX1 Compact」出來之後可以說是目前少有的小型旗艦機，有信仰的粉絲可以多多的支持。

言歸正傳，大部分人買完Android手機之後總是被其中的內置應用（預裝應用）所煩惱。如今很多預裝應用都可以disable，不過還是能在應用設置中找到，有些甚至都不能disable比如「Xperia」裡面的「What’s new」應用。本篇文章就是講述在不root的情況下如何清除這些內置應用，這個方法其實應該是應用於所有機型的，本篇文章題目純屬打廣告(:

下面先放出一些參考資料：

  • Sony Open devices
  • xda-developers “uninstall bloatware, method and which application remove”
  • xda-developers “[Q] Anyone managed to hide ‘What’s New’ app without root?”

操作

手機上的操作

對應用的處理

為了使刪除之後的殘留更少，我們首先要disable想要刪除的應用，然後delete cache和delete data。如果以上操作無法進行，最好也進行force stop來停止應用運行。在「Xperia」中，以上操作可以進入這裡找到 Settings >> Apps&notifications >> App info

打開開發者模式

然後就需要打開「開發者模式」了，在「Xperia」上，我們需要進入 Settings >> System >> About phone >> Build number，之後我們需要點擊「Build number」七次，從而打開「Developer options」。接下來進入其中，開啟「USE debugging」手機上的操作就完成了。

電腦上的操作

首先，去官方網站下載「SDK Platform Tools」解壓之後進入解壓後的路徑：

$ cd your/path/for/platform-tools

然後我們可以將手機與電腦連接，接下來在terminal中輸入以下命令來查看設備是否連接：

$ ./adb devices

如果連接會出現設備名和序列號，同時手機端也會有連接提示，點擊確認即可。

接下來輸入如下命令來查看deactivated的應用：

$ ./adb shell pm list packages -d

根據顯示出來的列表，我們可以找到之前被disable的應用，然後輸入如下命令刪除即可：

$ ./adb shell pm uninstall -k --user 0 <package name>

有些應用是不能被disable所以就不會出現在之前的list里，不過在上網查到它的名稱之後也可以用同樣的方法刪除，比如「Xperia」裡面的「What’s new」應用的名稱是com.sonymobile.entrance。筆者還總結了其它的可能需要刪除的包裹名稱，如下：

com.amazon.kindle
com.amazon.mShop.android.shopping
com.facebook.katana
com.facebook.services
com.facebook.system
com.facebook.appmanager
com.spotify.music
com.s.antivirus
com.sony.tvsideview.videoph
com.sonymobile.moviecreator.rmm
com.sonyericsson.xhs
com.google.android.videos
com.google.android.music

這樣我們不需要的內置應用就全部刪除啦～enjoy! (つ´ω`)つ
